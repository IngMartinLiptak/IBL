This solution is based on Angular v19. It uses only standard HTML and Angular controls.
This solution includes the implementation of a multi-input component with Input and Output parameters.

I prefer this solution — clean, simple, and lightweight.

This solution is based on the PrimeNG framework v19.
The components used are: AutoComplete, Checkbox, and Table.

When the use of a framework is necessary, I recommend using this one.

This framework is very robust, complex, and... difficult to use. Especially in cases where it is required to modify the standard look and behavior of controls.

I recommend using it only in large projects with standard control usage.
